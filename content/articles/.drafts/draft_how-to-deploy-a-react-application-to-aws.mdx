### Summary of tasks


1. Create a React App (if you don’t have one already)
2. Create a new AWS S3 bucket
3. Configure S3 bucket to host a Static Website
4. Configure settings and permissions
5. Manually publish React App to S3 then access it with public link
6. Bonus: Deploy your React app for CI/CD using AWS CDK
